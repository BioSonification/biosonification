## Ключевые концепции

**Фрагментированная генерация** — основной подход для длинных последовательностей:
- Входная последовательность разбивается на фрагменты по 1800 bp
- Для каждого фрагмента генерируется 4 или 8 тактов музыки
- Все фрагменты склеиваются в одну композицию
- Результат: стабильное качество независимо от длины последовательности

**Bio-Conditioned Transformer** — архитектура модели:
- Биологические признаки → 256D embedding
- Harmony model: генерирует аккордовую сетку по тактам
- Melody model: генерирует монофоническую мелодию поверх гармонии

**Доступные модели:**
- **4-тактовая модель** (рекомендуется): val_loss 0.145 (harmony), 0.157 (melody)
- **8-тактовая модель**: val_loss 0.179 (harmony), 0.215 (melody)
