## Ключевые концепции

### Фрагментированная генерация
Основной подход для работы с длинными последовательностями:
- Последовательность разбивается на фрагменты по 1800 bp
- Для каждого фрагмента генерируется 4 или 8 тактов
- Фрагменты объединяются в единую композицию
- Качество остается стабильным независимо от длины

### Архитектура модели
Bio-Conditioned Transformer с двухэтапной генерацией:
- Биологические признаки кодируются в 256D вектор
- Harmony model генерирует аккордовую сетку
- Melody model создает мелодию поверх гармонии

### Доступные модели
- 4-тактовая (рекомендуется): validation loss 0.145/0.157
- 8-тактовая: validation loss 0.179/0.215
